UDHËZUESI I SHPEJTË - QUICK START GUIDE

Ky është projekti juaj i ri doktoral me algoritme të sofistikuara PySpark!

================================================================================
HAPAT E SHPEJTË (3 HAPA TË THJESHTË)
================================================================================

HAPI 1: Instaloni libraritë
----------------------------
Hapni PowerShell dhe ekzekutoni:

    python -m pip install pyspark pandas matplotlib seaborn numpy openpyxl

Nëse Python nuk funksionon, shkarkoni nga: https://www.python.org/downloads/
(Zgjidhni "Add Python to PATH" gjatë instalimit)


HAPI 2: Gjeneroni dataset-in financiar
---------------------------------------
    python generate_dataset.py

Ky skedar do të krijojë një dataset financiar me ~2,470 rreshta (2015-2024)
File do të ruhet në: data_kaggle/financial_data.csv


HAPI 3: Ekzekutoni analizën
----------------------------
    python analiza_financiare_advanced.py

Ky skedar do të:
  ✓ Ngarkojë të dhënat
  ✓ Aplikojë të 10 algoritmet e profesorit
  ✓ Krijojë 5 CSV files me rezultate
  ✓ Krijojë 4 PNG figura profesionale

================================================================================
PËRMBAJTJA E KODIT (analiza_financiare_advanced.py)
================================================================================

Klasa Kryesore: AnalizaFinanciareAdvanced (650+ rreshta kodi)

Metodat:
  1. __init__()                      → Inicializimi i SparkSession
  2. ngarko_te_dhenat()             → Ngarkimi i CSV
  3. null_counting_analysis()       → Analiza e null values
  4. data_profiling_analysis()      → Profiling i plotë
  5. percentile_approx_analysis()   → Llogaritja e kuantileve
  6. returns_calculation()          → Daily, weekly, monthly returns
  7. moving_average_analysis()      → MA 7d, 30d, 90d
  8. volatility_computation()       → Historical & rolling volatility
  9. groupby_aggregation_analysis() → Monthly & yearly aggregations
  10. lag_time_series_analysis()    → Lag operations
  11. window_function_analysis()    → Rank, dense_rank, etc.
  12. ruaj_rezultatet()             → Ruajtja në CSV
  13. krijo_vizualizime()           → 4 figura profesionale
  14. mbyll()                       → Mbyllja e Spark

Pipeline i Plotë:
  main() → Ekzekuton të gjithë procesin nga fillimi deri në fund
